## 🎯 Design Elements

### Cards
- Glassmorphism effect with backdrop blur
- Border: 1px solid rgba(0, 212, 255, 0.1)
- Border radius: 12px
- Padding: 24px
- Hover effect: Lift with shadow

### Buttons
- Gradient background: #00d4ff to #0099cc
- Border radius: 8px
- Hover: Lift effect with enhanced shadow
- Text: Dark navy (#0a0e27)

### Charts
- Dark theme with neon colors
- Tooltips with dark background
- Grid lines: rgba(255,255,255,0.1)
- Smooth animations on render

### Typography
- Headings: 28-32px, weight 700
- Subtitles: 16-18px, color #8b92b0
- Body: 14-16px, color #c5cae9
- Labels: 12-14px, weight 500

### Animations
- Hover transitions: 0.3s ease
- Chart animations: 1s ease
- Page transitions: Smooth fade
- Pulsing orb: 4s infinite

## 📐 Layout Specifications

### Sidebar
- Width: 260px (open), 80px (closed)
- Background: rgba(15, 19, 41, 0.8)
- Fixed position
- Full height

### Main Content
- Margin-left: 260px (sidebar open)
- Padding: 30px
- Min-height: 100vh

### KPI Grid
- Grid: auto-fit, minmax(280px, 1fr)
- Gap: 20px
- Responsive: Stacks on mobile

### Charts Grid
- Grid: auto-fit, minmax(500px, 1fr)
- Gap: 20px
- Responsive: Single column on mobile

## 🎨 Visual Hierarchy

1. **Primary**: Large headings, KPI values, chart data
2. **Secondary**: Subtitles, labels, navigation items
3. **Tertiary**: Helper text, timestamps, metadata

**Contrast Ratios**: All text meets WCAG AA standards for readability

---

- **Desktop**: 1920px+ (full layout)
- **Laptop**: 1366px - 1920px (optimized)
- **Tablet**: 768px - 1366px (adapted grid)
