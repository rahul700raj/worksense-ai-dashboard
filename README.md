# WorkSense AI - HR Analytics Dashboard

Modern SaaS HR Analytics Dashboard with Dark Theme, Neon Blue Accents, and Enterprise Features.

## 🚀 Features

- **Multi-Role Dashboards**: HR Admin, Manager, and Employee views
- **AI-Powered Insights**: Intelligent analytics and predictions
- **Attrition Risk Analysis**: Predict and prevent employee turnover
- **Skill Gap Analysis**: Identify training needs and opportunities
- **Real-time Analytics**: Live KPIs and metrics
- **Responsive Design**: Mobile-first approach
- **Dark Theme**: Modern dark UI with neon blue accents

## 📊 Screens

1. **Login / Signup** - Secure authentication
2. **HR Admin Dashboard** - Complete workforce overview
3. **Manager Dashboard** - Team management and insights
4. **Employee Dashboard** - Personal metrics and goals
5. **AI Insights Page** - Machine learning predictions
6. **Attrition Risk Page** - Employee retention analytics
7. **Skill Gap Analysis** - Training and development tracking

## 🛠️ Tech Stack

- React 18
- Recharts for data visualization
- CSS3 with animations
- Responsive design

## 🎨 Design

- **Theme**: Dark + Neon Blue
- **Style**: Corporate + Futuristic
- **Typography**: Inter font family
- **Layout**: Modern SaaS dashboard

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build

```bash
npm run build
```
